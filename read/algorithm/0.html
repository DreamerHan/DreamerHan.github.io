# 二叉树

# 树的概念

树的概念在 DOM 结构中比较常见，比如如下代码中，常见的 DOM 树结构。

const tree = {
  val: "a",
  children: [
    {
      val: "b",
      children: [
        { val: "c", children: [] },
        { val: "d", children: [] },
      ],
    },
    {
      val: "e",
      children: [
        { val: "f", children: [] },
        { val: "g", children: [] },
      ],
    },
    {
      val: "h",
      children: [
        { val: "i", children: [] },
        { val: "j", children: [] },
        { val: "k", children: [] },
      ],
    },
  ],
};

树结构分为 “多叉树” 和 “二叉树”。

“多叉树” 结构如上的 DOM 树，根节点 a 有多个叶子节点 b e h。 h 节点又是一个多叉树。

“二叉树” 表示该节点之后两个叶子节点，被称为 左子树 和 右子树。类似于上面 DOM 树结构中 b 和 e 节点就是二叉树结构。

# 二叉树

一般表示二叉树的结构写法如下，分为 left 和 right 节点。

const tree = {
  val: "1",
  left: {
    val: "2",
    left: {
      val: "4",
      left: null,
      right: null,
    },
    right: {
      val: "5",
      left: null,
      right: null,
    },
  },
  right: {
    val: "3",
    left: {
      val: "6",
      left: null,
      right: null,
    },
    right: {
      val: "7",
      left: null,
      right: null,
    },
  },
};

二叉树作为树形结构中的重要类型，可以细分为多种不同类型：普通二叉树、满二叉树、完全二叉树、 二叉排序树（BST Binary Sort/Search Tree） 和 平衡二叉树（AVL 树）。

# 普通二叉树

普通二叉树是最基本的二叉树形式。

如下图，E 节点可有可无。

其特点是它并不要求每个节点都有两个字节点。因此，一个普通二叉树的每一层可并不是结构完善的。同时，它的叶子节点可能也不再底层，并且肯能存在空隙的部分。

          a
       b       d
    c       e    f
 c1   c2

如上一个普通的二叉树，字节点 b 只有左子树，并没有 右子树。所以该二叉树是有空隙的部分的。

同时，e 和 f 两个叶子节点也并不在最底层。

# 满二叉树

满二叉树和普通二叉树的主要区别体现在它们的结构特点和结点数上。

满二叉树是一种特殊的二叉树，其特点在于每一层的结点数都达到最大值。

所以，满二叉树突出的是 “满”，每一层都是满的，没有空缺。

它的每个非叶子结点的度都为 2，即每个结点都有两个子结点，不存在度为 1 的结点。

这使得满二叉树的叶子节点只能出现在层次最大的两层上，并且整个树的形状非常规整，类似于一个三角形。

满二叉树计算公式，假设满二叉树的深度为 K：

  1. 它的结点总数将是 (2^k) - 1
  2. 它的最后一层节点数是 2^(k-1)
  3. K = log2(节点总数 + 1)

# 完全二叉树

如果二叉树除了最后一层有缺失外，其它是满的，且最后一层缺失的叶子结点只出现在右侧，则这样的二叉树叫完全二叉树。

假如二叉树的深度为 n，除第 n 层外，其余各层的结点都达到了最大，且第 n 层的结点都连续集中在最左边。(完全二叉树要求叶子节点从左往右填充，填充满后才能开始填充下一层。)

简而言之，就是从左到右结点是连续不断的二叉树就叫完全二叉树。

# 二叉排序树

又称 “BST Binary Sort/Search Tree”

二叉排序树要求对于任何一个非叶子节点，其左子节点的值都小于当前节点的值，而右子节点的值都大于当前节点的值。这使得二叉排序树在搜索和排序操作中非常高效。

# 平衡二叉树（AVL 树）

平衡二叉树是一种特殊的二叉排序树，其特点是任何节点的两个子树的高度差的绝对值不超过 1，且左右两个子树都是平衡二叉树。平衡二叉树的常用实现方法有红黑树、AVL、替罪羊树、Treap、伸展树等。

# 深度/广度优先遍历

假设一个树结构如下：

const tree = {
  val: "a",
  children: [
    {
      val: "b",
      children: [
        { val: "c", children: [] },
        { val: "d", children: [] },
      ],
    },
    {
      val: "e",
      children: [
        { val: "f", children: [] },
        { val: "g", children: [] },
      ],
    },
  ],
};

# 深度遍历

深度优先遍历就是从根节点，深入到一个根节点的一个子节点，然后一直遍历到该子节点的最深节点。然后再返回遍历根节点的另外一个子节点，直到最深节点，依次轮回。

针对上述树结构，深度遍历的结果是：

a -> b -> c -> d -> e -> f -> g

实现方式就是直接对存在 chilren 进行遍历即可：

const deep = (root) => {
  console.log("深度遍历", root.val);
  root.children.forEach((item) => deep(item));
};

deep(tree);

# 广度遍历

广度遍历是按树结构的层级进行遍历的，它先摆平完整的一整个层级，然后再继续深入更深层级。

针对上述树结构，遍历结果如下： a -> b -> e -> c -> d -> f -> g

实现技巧，可以通过队列的形式，先进先出。从根节点出发，优先访问距离根节点最近的节点。

// 1. 创建一个队列，先把根节点入队
// 2. 队头出对
// 3. 把队头的 children 挨个入队
// 4. 重复 2 和 3，知道队列为空

const space = (tree) => {
  const arr = [tree];

  while (arr.length > 0) {
    o = arr.shift();

    console.log(o.val);

    o.children.forEach((item) => arr.push(item));
  }
};

console.log(space(tree));

# 前序遍历

二叉树的 前序遍历又称 先序遍历。

它的遍历技巧是：左右左。任何一级都是先左后右，同时，每个子树再进行深度遍历。

针对上述数据结构，遍历的数组结果为：1 -> 2 -> 4 -> 8 -> 9 -> 5 -> 10 -> 11 -> 3 -> 6 -> 12 -> 13 -> 7 -> 14

# 递归方式实现如下

const preOrder = (tree) => {
  let arr = [];

  const fun = (node) => {
    if (node) {
      arr.push(node.val);

      node.left && fun(node.left);
      node.right && fun(node.right);
    }
  };

  fun(tree);

  return arr;
};

console.log(preOrder(tree));

# 栈模式实现如下

const stackMode = (tree) => {
  let arr = [];

  if (!tree) {
    return arr;
  }

  // 根节点入栈
  let stack = [tree];
  while (stack.length) {
    let o = stack.shift(); // 把头的出栈，配合下面的 unshift
    arr.push(o.val);

    // 这里是最有意思的，让 right 先进栈，但是通过 unshift 方式
    o.right && stack.unshift(o.right);

    // 当 left 再进栈时，是放在了 right 的前面。所以，当 shift 出栈时，left 的是后进先出
    o.left && stack.unshift(o.left);
  }

  return arr;
};

console.log(stackMode(tree));

# 中序遍历

二叉树的 “中序遍历”，遍历技巧：左根右

它优先深入到 “左子树” 的最深一层的 “左子树”，然后向上一级找 ”左子树根“，再从 ”左子树根“ 找它的 ”右子树“。 左侧都遍历结束后，开始”右子树“的最深一层的 “左子树”，然后向上一级找根，再找根的 “右子树”。

针对上述数据结构，遍历的数组结果为：4 -> 2 -> 5 -> 1 -> 6 -> 3 -> 7

# 递归方式实现如下

递归的注意点是，要先递归 左子树 left 的各项，然后中间才 push 根节点的 val，最后再递归右子树 right 的各项。

const deepModeTest = (tree) => {
  let arr = [];

  const fun = (node) => {
    node.left && fun(node.left);

    arr.push(node.val);

    node.right && fun(node.right);
  };

  fun(tree);

  return arr;
};

console.log(deepModeTest(tree));

# 栈模式实现如下

const stackMode = (tree) => {
  const arr = [];
  const stack = [];

  let o = tree;
  while (stack.length || o) {
    while (o) {
      stack.push(o);
      o = o.left;
    }

    const n = stack.pop();
    arr.push(n.val);

    o = n.right;
  }

  return arr;
};

stackMode(tree);

# 后序遍历

二叉树的 “后序遍历”，遍历技巧：左右根

它优先深入到 “左子树” 的最深一层的 “左子树”，然后寻找 “右子树”，最后向上寻找上一级。 左侧都遍历结束后，开始”右子树“的最深一层的 “左子树”，然后寻找"右子树"，再向上一级寻找根。

针对上述数据结构，遍历的数组结果为：4 -> 5 -> 2 -> 6 -> 3 -> 7 -> 1

# 递归模式

const deepMode = (tree) => {
  const arr = [];

  const fun = (node) => {
    if (!node) return;

    fun(node.left);
    fun(node.right);
    arr.push(node.val);
  };
  fun(tree);

  return arr;
};

console.log("递归模式", deepMode(tree));

# 栈模式

const stackModeTest = (tree) => {
  const arr = [];
  const stack = [tree];

  while (stack.length) {
    const o = stack.pop();
    arr.unshift(o.val);

    o.left && stack.push(o.left);

    o.right && stack.push(o.right);
  }

  return arr;
};

console.log(stackModeTest(tree));

# 最小深度

最小深度的遍历是针对层级的遍历。

满足最小深度的层级，一定是提前出线了 node.left 和 node.right 同时不存在的一级。否则，就需要增加层级深度，继续遍历下去。

下述方法最需要注意的是 shift 和 push 的结合处理，遍历就是最简单的从上到下遍历，有 left 就 push left，有 right 就 push right。

push 的方式，就使得子级都在父级后面。所以，层级遍历时，就要使用 shift 从前抛出，先走父级，再走子级。

const miniDepth = (tree) => {
  if (!tree) return 0;

  const stack = [[tree, 1]];

  while (stack.length) {
    const [o, length] = stack.shift();

    if (!o.left && !o.right) {
      return length;
    }

    o.left && stack.push([o.left, length + 1]);
    o.right && stack.push([o.right, length + 1]);
  }
};

console.log(miniDepth(tree));

# 最大深度

最大深度是寻找当前树的最多层级。所以，循环的技巧是 一层一层 的如栈，将一层的全部节点如栈后，再进行全部栈。

出栈的同时，将出栈的下一层节点内容继续如栈，然后再轮回出栈。

在每次栈内是否还有内容进行循环时，对节点层级叠加。知道没有内容再如栈为止。

const maxDepth = (root) => {
  let num = 0;

  const stack = [tree];

  while (stack.length) {
    let length = stack.length;

    // 叠加一层
    num++;

    // 接着将当前层的内容全部出栈
    while (length--) {
      const o = stack.shift();

      // 当前层的下一级如栈
      o.left && stack.push(o.left);
      o.right && stack.push(o.right);
    }
  }

  return num;
};

反转二叉树，就是将根节点下的子节点形成镜子树结构，说白了就是父节点下左右节点互换。

所有位置互换的问题，都是创建一个第三方 C，然后让 A 赋值给 C，让 B 赋值给 A，最后让 C 赋值给 B。

对于树的问题，就是在互换的操作中不断递归就好了。

# 递归模式

const reverseTree = (tree) => {
  if (tree === null) return null;

  let tmp = tree.left;
  tree.left = tree.right;
  tree.right = tmp;

  reverseTree(tree.left);
  reverseTree(tree.right);

  return tree;
};

console.log(reverseTree(tree));

# 栈模式

const stackMode = (tree) => {
  if (!tree) return;

  const stack = [tree];

  while (stack.length) {
    const o = stack.shift();

    const node = o.left;
    o.left = o.right;
    o.right = node;

    o.left && stack.push(o.left);
    o.right && stack.push(o.right);
  }

  return tree;
};

console.log("栈模式", stackMode(tree));

const isSameTree = (tree1, tree2) => {
  if (tree1 === null && tree2 === null) {
    return true;
  }

  if (tree1 === null || tree2 === null) {
    return false;
  }

  if (tree1.val !== tree2.val) {
    return false;
  }

  return (

  );
};
