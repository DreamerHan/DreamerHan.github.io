# 堆

堆，是一种特殊的完全二叉树。

但是对和树的数据结构形式不同，树的结构一般都利用类似于链表的形式，通过 chilren 或是 left 又或是 right 等字段扩展下一级，类似于原型链中的 __proto__。

但是堆不同，堆只能说结构像树，但实际上通过数据形式来实现表示时，它是用数组的形式展示的。

# 堆类型

  1. 常用的堆有两种：

    • 最大堆： 所有的节点都 大于等于 ≥ 它的子节点。
    • 最小堆： 所有的节点都 小于等于 ≤ 它的子节点。

  2. 堆有三个公式：

  • 左侧子节点位置是 2 * 数组索引 index + 1
  • 右侧字节点位置是 2 * 数组索引 index + 2
  • 父节点的位置是 Math.floor((数组索引 index - 1) / 2)

# 最小堆

最小堆是一种经过排序的完全二叉树，其中任意非终端节点数值均不大于其左子节点和右子节点的值。就是说，最小堆，是根节点永远比左右子节点都小。
