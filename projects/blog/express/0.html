# 开启 express

# express-generator

通过安装 express-generator 可以安装一个全局的 express 命令，然后通过 express 项目名称 即可开始创建一个 express 的项目

npm install express-generator -g --registry=淘宝镜像  即可

express 项目名称

npm install 安装依赖

npm start 启动服务

# 原生项目结构

脚手架本身生成的项目结构，是一个类似混合式开发的全栈项目结构。类似于 public 与 views 等文件夹，都是在需要开发前端视图时才用的到的，目前可以忽略。

  .
  ├── app.js  // 入口文件
  ├── bin
  │   └── www  // 服务启动脚本文件
  ├── package.json
  ├── public   // 公共文件
  │   ├── images
  │   ├── javascripts
  │   └── stylesheets
  │       └── style.css
  ├── routes  // 路由
  │   ├── index.js
  │   └── users.js
  ├── views // 视图
  │   ├── error.jade
  │   ├── index.jade
  │   └── layout.jade
  └── yarn.lock

// 安装相关依赖
