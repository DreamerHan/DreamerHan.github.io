# 简单的 sql 语句

插入语句：insert into 表名 (key1, key2, `passwod`) value (value1, value2, value3);

查询语句：
  - 全部查询： select * from 表名;
  - 字段查询： select key1, key2 from 表名;
  - 模糊查询： select * from 表名 where key like '%模糊字段%';
  - 条件排序： select * from 表名 where key=value order by key名;

更新语句：update 表名 set key=value where key=value;

删除语句：
  - delete from 表名 where key=value;
  - 正常开发工作，都是软删除，比如更改一下该条的删除状态，而不是真的删除

版本查询：select version();

备注：
  - 像 password这种保留字要添加 `` 进行包裹
  - 不等于符号是 <>
  - asc 升序; desc 降序(order by key desc)
  - varchar(10) Mysql 版本大于5已上，varchar(n)中的 n 表示字符为 n个，无论是数字、英文或是汉字
  - sql 语句的注释方式是在 sql 语句前面加两个 - 和一个宫格，即： `-- `
