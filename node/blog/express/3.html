# 项目结构搭建

express 的项目搭建主要也是参考 node 版本的形式，在 express 现有的项目结构上，添加 mysql、redis、controller、model数据模型、utils 公共方法 以及公共配置文件 config 即可

TIP

安装依赖：yarn add mysql redis@3.1.2 xss -S

redis 由于升级到 4.0 版本之后就变样子了，还需要手动执行 client.connect() 链接服务，导致与 express-session 结合起来特别麻烦，没研究明白。索性放弃安装了 4.0 以下的版本

坐等大神的标准格式写法吧 (๑◕︵◕๑)...
