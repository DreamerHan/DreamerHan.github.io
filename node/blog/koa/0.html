# 开启 koa

# koa-generator

和 express一样，可以通过安装 koa-generator 安装一个全局的 koa 命令，然后通过 koa2 项目名称 即可开始创建一个 koa 的项目

npm install koa-generator -g --registry=淘宝镜像  即可

koa2 项目名称

npm install 安装依赖

npm run dev 启动服务

# 原生项目结构

脚手架本身生成的项目结构，同样是一个类似混合式开发的全栈项目结构。类似于 public 与 views 等文件夹，都是在需要开发前端视图时才用的到的，目前可以忽略。

  .
  ├── app.js
  ├── bin
  │   └── www
  ├── package.json
  ├── public
  │   ├── images
  │   ├── javascripts
  │   └── stylesheets
  │       └── style.css
  ├── routes
  │   ├── index.js
  │   └── users.js
  ├── views
  │   ├── error.pug
  │   ├── index.pug
  │   └── layout.pug
  └── yarn.lock
