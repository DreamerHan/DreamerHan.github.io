# 简略

该博客项目是一个简单的接口实现流程展示项目

它包含了使用 nodeJs 原生模块、express 框架、koa 框架三种实现方式

后面还使用了 mongodb 来重构了一次数据库的实现方式

  • 当前 node 版本号 v14.17.3

  • 重点说明之 redis :

    redis 依赖这破玩意，redis 4.0 版本以后，创建客户端好像变了，需要手动执行 await client.connect() 进行客户端与服务端的链接。

    但是，链接过程是异步的，导致与 express-session 结合时没捣鼓明白，所以 express 版本特意安装了 4.0 以下的版本

    node 版本写法时，由于当时头刚非要用 4.0.1 的版本，写了异步的方式，不过勉强能看，但方式肯定还得优化
